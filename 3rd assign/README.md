# Assignment 3: ShopVibe Frontend

The frontend implementation of "**ShopVibe**", a modern e-commerce platform built using React and Vite.

## Tech Stack
- **React 18** with **Vite**
- **Tailwind CSS v4** for styling
- **React Router v6** for navigation
- **Axios** for API calls
- **Lucide React** for icons
- **React Hot Toast** for notifications

## Getting Started
1. Open terminal and navigate to the `frontend` folder:
```bash
cd frontend
```
2. Install the necessary dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm run dev
```
