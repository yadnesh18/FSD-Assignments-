# Assignment 2: Personal Portfolio Website

A complete, responsive personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## Features
- **Responsive Navigation:** Fixed navbar with scroll observation to highlight the active section link.
- **Hero Section:** Features a unique custom "code window" animation.
- **About, Skills, Work & Contact Sections:** Grid-based layouts showcasing skills, achievements, and links.
- **Animations:** Uses `ScrollReveal.js` for smooth element revealing while scrolling.
- **Styling:** Custom CSS with styling inspired by Tailwind design concepts.

## How to use
Simply open `Portfolio/index.html` in your web browser.
