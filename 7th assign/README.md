# Assignment 7: Student Feedback Review System

A full-stack project built with the MERN stack (MongoDB, Express, React, Node.js) designed to collect and display student feedback effectively. 

## Architecture
- **Frontend**: A React (Vite) client providing a submission form layout and a dashboard to list existing feedback responses.
- **Backend**: An Express.js REST API resolving CORS efficiently and integrating with a MongoDB database to store feedback permanently.

## Getting Started

### Backend
1. Go to the `Backend/` directory: `cd Backend`
2. Install dependencies: `npm install`
3. Set your MongoDB details inside `.env`.
4. Run the backend server: `npm run dev`

### Frontend
1. Go to the `Frontend/` directory: `cd Frontend`
2. Install dependencies: `npm install`
3. Run the frontend: `npm run dev`
