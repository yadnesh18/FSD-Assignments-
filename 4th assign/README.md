# Assignment 4: Interactive Weather Dashboard

A React-based interactive weather dashboard built utilizing highly customized SVG data visualizations.

## Features
- Custom **SVG Charts** built entirely from scratch without the use of external charting libraries like `Recharts` or `Chart.js`.
- Dual-axis line charts (Temperature & Humidity).
- Bar charts (Rain Probability, UV index).
- Donut charts (Weather conditions distributions).
- Detailed mockup logic directly within `App.jsx`.
- Clean, minimal, dashboard-style UI.

## Getting Started
1. Open terminal and navigate to the `vite-project` directory:
```bash
cd vite-project
```
2. Install the necessary dependencies:
```bash
npm install
```
3. Start the Vite development server:
```bash
npm run dev
```
