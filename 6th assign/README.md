# Assignment 6: CampusEscapes - College Travel Agency

A full-stack travel agency web application decoupled into a RESTful architecture, allowing college students to browse and book trips.

## Architecture
- **Frontend**: A React application utilizing Vite, located in the `Frontend/` folder. Contains Home, Book, and Success route flows.
- **Backend**: An Express.js REST API server located in the `Backend/` folder. Handles database interactions and serves JSON data.

## Getting Started

### Backend
1. Open terminal and navigate to `Backend/`.
2. Install dependencies: `npm install`
3. Set your environment variables in a `.env` file based on `.env.example`.
4. Run the server: `npm run dev`

### Frontend
1. Open terminal and navigate to `Frontend/`.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
