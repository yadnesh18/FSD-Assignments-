# Assignment 1: HTML Text Formatting Tags

A fundamental HTML assignment demonstrating the use of various text formatting tags (`<b>`, `<strong>`, `<i>`, `<em>`, `<mark>`, `<small>`, `<del>`, `<ins>`, `<sub>`, `<sup>`).

## Features
- Structured inside a styled HTML table.
- Demonstrates semantic vs physical HTML tags.
- Simple offline `index.html` file setup.

## How to use
Simply double-click on `1st.html` to open it in your web browser.
