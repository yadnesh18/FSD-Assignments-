# Full Stack Development Assignments

This repository contains a collection of 7 assignments focused on Web Development, progressing from basic HTML formatting to full-stack MERN applications.

## 📁 Assignments Overview

### 1. [HTML Text Formatting Tags](./1st%20assign)
A fundamental HTML assignment demonstrating the use of various text formatting tags (`<b>`, `<strong>`, `<i>`, `<em>`, `<mark>`, `<small>`, `<del>`, `<ins>`, `<sub>`, `<sup>`) organized within a styled HTML table.

### 2. [Personal Portfolio Website](./2nd%20assign)
A responsive personal portfolio website built with HTML, CSS (Tailwind CSS conceptually), and vanilla JavaScript.
- **Features:** Navigation bar with active link highlighting, coding animation header, About, Skills, Work, and Contact sections, along with smooth scrolling and ScrollReveal animations.

### 3. [ShopVibe Frontend](./3rd%20assign)
The frontend implementation of "**ShopVibe**", a modern e-commerce platform built using React and Vite.
- **Tech Stack:** React 18, Vite, Tailwind CSS v4, React Router v6, Axios, Lucide React.
- **Features:** This serves as the UI layer for the full-stack e-commerce project built later.

### 4. [Interactive Weather Dashboard](./4th%20assign)
A React-based interactive weather dashboard built with Vite. 
- **Features:** Highly customized SVG data visualizations including dual-axis line charts (Temperature & Humidity), bar charts (Rain Probability, UV index), and donut charts (Conditions distributions), built from scratch without external charting libraries.

### 5. [ShopVibe - Full-Stack E-Commerce](./5th%20assign)
A complete, production-ready e-commerce application built on the MERN stack (MongoDB, Express, React, Node.js).
- **Features:** Modern UI with dark mode, JWT authentication, role-based access control, a full admin dashboard (product and order management), cart functionalities, and checkout flow.
- **Architecture:** Decoupled frontend (React) and backend (Express) with RESTful APIs.

### 6. [CampusEscapes - College Travel Agency](./6th%20assign)
A web application providing trip booking functionalities for students.
- **Architecture:** Restructured from a monolithic layout into a decoupled architecture with a React (Vite) frontend and an Express REST API backend. 
- **Features:** Home page listing trips, booking form routes, and success pages.

### 7. [Student Feedback Review System](./7th%20assign)
A full-stack MERN application for collecting and displaying student feedback to assess performance.
- **Architecture:** Decoupled architecture with a React frontend and Express backend APIs communicating on different ports, backed by a NoSQL database.
- **Features:** Submission forms and dynamic feedback list screens.

## 🚀 Getting Started

To run the React or Full-Stack projects (Assignments 3 through 7):
1. Navigate into the specific assignment directory's `Frontend` or `backend` folder.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
